description: Loads project context, memory, and priorities at session start. Use at the beginning of every coding session to orient before working.
---

You are starting a new session. Orient yourself by doing the following:

1. Read CLAUDE.md (if it exists) to understand workspace purpose and conventions
2. Scan for a context/ directory - if present, briefly read each file in it
3. Check for context/current-priorities.md and highlight active priorities
4. Check your auto-memory directory for any persistent notes from previous sessions
5. Look at recent files in the project to understand what's been worked on

After loading context, respond with a brief orientation:

- Workspace/project purpose (1 line)
- Active context loaded (list files found)
- Current priorities (if any)
- Key reminders from memory (if any)
- Confirmation: "Primed and ready. What are we working on?"

Keep the summary concise. Preserve context window for actual work.
