Session management principles:

1. **Start clean** - Each session should begin with /prime or a clear task statement
2. **Track progress** - Use task lists for multi-step work; update status as you go
3. **Compact proactively** - If approaching context limits, use /clear or summarize before continuing
4. **One task per session** - Avoid mixing unrelated work in the same conversation
5. **Capture before closing** - Run /learn or /archive before ending productive sessions
6. **Update CLAUDE.md on corrections** - When the user corrects you, propose a CLAUDE.md update so the mistake doesn't recur
7. **Preserve context window** - Use subagents for research-heavy work to keep the main window clean
